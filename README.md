## Code Conventions

Use interfaces instead of classes. For example,
```ts
interface Bag<T> {}

interface DisjointUnion<T> extends Bag<T> {
    a: Bag<T>;
    b: Bag<T>;
}

function disjointUnion<T>(a: Bag<T>, b: Bag<T>): DisjointUnion<T> {
    return {
        a: a,
        b: b,
    };
};
```
